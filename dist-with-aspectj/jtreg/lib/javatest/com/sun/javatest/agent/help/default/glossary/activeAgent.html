Active Agent

An agent that initiates a connection to the JT Harness harness.

Active agents enable you to run tests in parallel using many agents at once and to specify the test machines at the time you run the tests. Use the Agent Monitor window to view the list of registered active agents and synchronize active agents with the JT Harness harness before running tests.
