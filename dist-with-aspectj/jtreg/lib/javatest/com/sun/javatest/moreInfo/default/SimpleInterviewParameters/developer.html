Developer mode has been specified for this interview by invoking the harness using the following command-line option:
-DSimpleInterviewParameters.mode=developer

This causes the tests to be compiled and executed when the test suite is run.

To compile the tests, you must also set command.compile.java system property using a command-line option. See the "Compiling Test Suites" appendix in the JT Harness Architect's Guide for details.
