The JT Harness Configuration Editor interviews you about the parameters used to run your tests. Test parameters are values that determine which tests in the test suite are run and how they are run. The interview contains between 10 and 14 questions.

The harness saves your parameter settings in the work directory so that they can be used with subsequent test runs. You can save your answers at any time using the commands in the File menu.

Please keep this More Info pane open the first few times you use the editor; it contains important information and examples about each question. To close (and reopen) this More Info pane click in the Question pane.

For more information about how to use the editor, press the F1 key or choose Help > JT Harness Configuration Editor from the editor menu bar.

To begin the interview click in the Question pane.
