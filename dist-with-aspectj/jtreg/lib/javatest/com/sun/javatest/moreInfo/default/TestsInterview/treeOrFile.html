The configuration editor provides two means of specifying the tests or folders of tests that are executed:
  • You can use an editor panel with which you can directly choose the tests and test branches for execution.
  • You can specify a text file containing a list of tests or sections of the test suite you wish to run.

If you answer "Directly," you will be presented with a tool in the next editor panel with which you can choose tests and test branches for execution.

If you answer "From a List in a File," you will be presented with a field in the next the editor panel with which you can provide the location and name of the file containing the list of tests or sections of the test suite you wish to run.
