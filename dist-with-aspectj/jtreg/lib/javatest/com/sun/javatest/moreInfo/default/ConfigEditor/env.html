Use this panel to create a list of environment files. Environment files contain test environments that the harness uses to run tests in your computing environment.

Click Add to activate a file chooser with which you select environment files (.jte). As you make selections they are added to the list. After you have added selections, you can modify the list as follows:
Remove Select an item in the list and click Remove
Move
Up 		Select an item in the list and click Move Up to move it up the list
Move
Down 		Select an item in the list and click Move Down to move it down the list

Use the Name combo box to choose one of the test environments contained in the environment file.
