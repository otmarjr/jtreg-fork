Setting Preferences

Choose File > Preferences from the menu bar to open the harness preferences dialog. Use this dialog to set the display and functional options of the harness.

The following table describes the contents of the Preferences dialog box.

Preference Category Description
Appearance Sets how the desktop displays tool windows and tool tips. See Appearance Preferences for detailed information.
Color Preferences Sets the color preferences for default input and input status warnings. See Color Preferences for detailed information.
Test Manager Sets the Test Manager tool bar property. See Test Manager Preferences for detailed information.
