Viewing Test Suite Errors

When the harness detects errors in the test suite, it displays an error dialog box and enables the View > Test Suite Errors menu item.

To display the Test Suite Errors viewer, you can choose View > Test Suite Errors from the Test Manager menu bar.

The dialog box displays a list of errors detected in the test suite. Unless instructed otherwise, report any test suite errors to the owner of the test suite.
