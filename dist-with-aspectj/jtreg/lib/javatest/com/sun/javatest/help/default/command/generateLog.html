Generating a Log of All Updates

You can use the -l option to generate a log of all updates to the .jti file which can be used later.

Example:
java -cp [jt_dir /lib/] javatest.jar com.sun.javatest.EditJTI -o mynew.jti -l myeditlog.html "|/java/jdk/1.3/|/java/jck/1.4/|" myoriginal.jti
