Files Pane

The Files tab displays those files nominated by a test suite on a per-test basis. The Files tab can render .html, .rtf, and .txt files as well as most .au, .wav, .mid, .gif, .jpg, and .png files.You can browse but not edit source files in this panel.

Choose a file from the drop-down list to display its contents.
