Multi-Selection Panel

When multiple tests are selected in the test tree, the harness displays a Multi-Selection panel in the Information Area.

  1. Test Tree Selection
  2. Multi-Selection Panel

This panel contains a text description area at the top of the panel that describes the operations that can be performed using tests and folders selected in the test tree and listed below.

In the section below the text description area, the harness displays a list of the tests and folders selected in the test tree.
